## Cách push code lên github
### Tạo repo trên github
###  Đẩy code lên repo
#### Khởi tạo một repo
```
git init
```
#### Liên kết repo
```
git remote add origin <URL của Repository GitHub>
```
#### Thêm tất cả các tệp và thư mục trong thư mục của bạn vào Repository
```
git add .
```
#### Commit các thay đổi
```
git commit -m 'Commit'
```
#### Đẩy code lên github
```
git push -u ogirin <Branch>
```
